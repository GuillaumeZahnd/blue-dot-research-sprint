# Tamper Attack Resistance

## Installation & Setup

**1. Install Pipenv (if not already installed):**

```sh
pip install --user pipenv
```

**2. Initialize environment:**

```sh
pipenv install --dev --python 3.12
```

**3. Activate environment:**

```sh
pipenv shell
```
